About this application

For the course "Visualisierung 2" we implemented a 2D spatial distribution visualization based on the approach presented in the paper "Phoenixmap: An Abstract Approach to Visualize 2D Spatial Distributions" [1].
We chose to visualize AirBnb listings in Paris from a public kaggle dataset [2]. Multiple views were implemented allowing display of overlayed Phoenixmaps generated with different settings. We experimented with using a convex hull (as proposed in the paper) as well as the districts actual border (derived from public shapefiles [3]) as outline.
